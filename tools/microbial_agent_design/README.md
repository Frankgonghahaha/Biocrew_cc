# 微生物菌剂设计工具集

本目录包含了用于微生物菌剂设计的一系列专业工具。

## 工具列表

1. **GenomeSPOT工具** - 用于预测微生物的环境适应性特征
2. **DLkcat工具** - 用于预测酶催化速率
3. **Carveme工具** - 用于构建基因组规模代谢模型
4. **Phylomint工具** - 用于分析微生物间的互作关系
5. **ctFBA工具** - 用于计算微生物群落的代谢通量
