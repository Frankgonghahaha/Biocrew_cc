# 菌剂设计工具集

本目录包含了用于微生物菌剂设计的各种工具，这些工具已成功集成到菌剂设计智能体中。

## 工具列表

### 1. GenomeSPOT工具
- **功能**：预测微生物的环境适应性特征
- **预测指标**：温度、pH、盐度和氧气耐受性
- **集成状态**：✅ 已集成

### 2. DLkcat工具
- **功能**：预测降解酶对于特定底物的降解速率
- **预测指标**：Kcat值（酶催化常数）
- **集成状态**：✅ 已集成

### 3. Carveme工具
- **功能**：构建基因组规模代谢模型(GSMM)
- **应用**：用于ctFBA算法计算代谢通量
- **集成状态**：✅ 已集成

### 4. Phylomint工具
- **功能**：分析微生物间的代谢互补性和竞争性
- **应用**：评估菌剂群落的稳定性和协同作用
- **集成状态**：✅ 已集成

### 5. ctFBA工具
- **功能**：计算微生物群落的代谢通量
- **核心算法**：协同权衡代谢通量平衡法
- **集成状态**：✅ 已集成
